# 💼 HTML Resume Layout

A simple and clean personal resume layout built with only HTML and CSS.  
This project was created as part of a Git & GitHub Pages assignment.

---

## 🔗 Live Demo

👉 [View the resume](https://quintanbarnes.github.io/homepage/)

---

## 🧩 Features

- Semantic HTML5 markup  
- Responsive layout using media queries  
- Clean and structured CSS  
- Avatar, contact info, skills, experience, education  
- Valid and accessible HTML

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- Git / GitHub / GitHub Pages

---

## 🚀 GitFlow & Structure

This project was developed in a separate feature branch:

- `feature/body` → merged into → `main`

All changes were committed with clear messages and integrated through a Pull Request.

---

## 📁 Project Management

Milestones:
- Create HTML structure  
- Add styles via `styles.css`  
- Add avatar and assets  
- Finalize layout and polish UX  
- Merge PR and publish via GitHub Pages
